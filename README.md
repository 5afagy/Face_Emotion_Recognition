## Dataset
<b>Dataset Link:</b> https://www.kaggle.com/deadskull7/fer2013 <br>
The dataset contains 35887 images of people showing 7 unique emotions: Anger, Happy, Disgust, Sad, Surprise, Neutral and Fear. <br>
